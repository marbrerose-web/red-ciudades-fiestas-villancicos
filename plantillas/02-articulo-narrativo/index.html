Proyecto de visualización de datos

Red de ciudades y fiestas en un corpus de villancicos hispanoamericanos

Una red bipartita permite observar cómo se distribuyen las fiestas y advocaciones religiosas en un corpus de villancicos hispanoamericanos, y qué ciudades concentran repertorios devocionales compartidos o perfiles más específicos.

Carolina — 26.04.2026

Esta visualización parte de una pregunta sencilla: ¿cómo se conectan las ciudades registradas en el corpus con las fiestas y advocaciones que aparecen en los villancicos? La red permite mirar el conjunto no como una lista de obras aisladas, sino como un sistema de relaciones entre lugares, devociones y repertorios.

Cada nodo naranja representa una ciudad y cada nodo azul representa una fiesta o advocación. Las líneas indican la presencia de una relación entre ambos elementos. De este modo, la gráfica hace visibles zonas de concentración, coincidencias devocionales entre ciudades y casos más singulares dentro del corpus

Después presenta el alcance de los datos: periodo, territorio, unidades de análisis y límites. Esta parte no debe sentirse como una ficha técnica aislada, sino como el marco de lectura del proyecto.
